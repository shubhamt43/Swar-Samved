## Quick Command Reference

```bash
# Frontend setup and run
pnpm install && pnpm dev         # http://localhost:3000

# Backend setup and run
cd backend && uv init --bare . && \
uv add fastapi uvicorn python-multipart librosa numpy scipy pydantic && \
uv run main.py                   # http://localhost:8000

# Test backend health
curl http://localhost:8000/health

# Development workflow
# Terminal 1: pnpm dev (frontend)
# Terminal 2: cd backend && uv run main.py (backend)
# Terminal 3: Optional - any additional commands

# Clean up and restart
rm -rf node_modules pnpm-lock.yaml     # Frontend clean
cd backend && rm -rf .venv __pycache__  # Backend clean
```
